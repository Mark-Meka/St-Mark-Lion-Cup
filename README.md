# St Mark Lions Cup ⚽🦁

A Church Soccer League management app — track standings, record match results, and manage teams.

## 🚀 Deploy to GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at `https://<your-username>.github.io/St-Mark-Lion-Cup/`

## 💻 Run Locally

Just open `index.html` in your browser — no server needed!

Or use any static file server:

```bash
npx serve .
```

## 📝 Features

- **League Standings** — auto-calculated points, goal difference, wins/draws/losses
- **Add Match Results** — select teams and enter scores
- **Delete Matches** — remove wrong results, standings auto-recalculate
- **Rename Teams** — customize team names anytime
- **All data stored in browser localStorage** — persists across page refreshes

## ⚠️ Note

Data is stored in your browser's localStorage. It is per-device and will be lost if you clear browser data.
